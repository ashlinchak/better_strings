# BetterStrings

This is a gem for adding more functionality for Ruby String class

## Installation

Add this line to your application's Gemfile:

``` ruby
gem 'better_strings'
```

And then execute:

```
$ bundle
```

Or install it yourself as:
    
```
$ gem install better_strings
```
